# EDI Notepad 2026

The modern successor to the Liaison EDI Notepad — a browser-based EDI inspector
for analysts, support engineers, and integration developers.

## Status

Phase 1: rebrand and scope reduction. The app is a single-purpose Notepad
(no AI, no other tabs). Phase 2+ will expand the segment/element dictionary
across Supply Chain, Logistics, Retail, CPG, Manufacturing, Grocery & Cold
Chain, and Financial Services, plus rebuild the classic Notepad utilities
(multi-view, ACK generation, control-number incrementing, find/print, single-
interchange export, build/edit, multi-doc tabs, TRADACOMS).

## Run locally

```
npm install
npm run dev
```

Open http://localhost:3000/edi-notepad.

## Test

```
npm test
```
